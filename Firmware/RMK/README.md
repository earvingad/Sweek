# Intructions for RMK

RMK firmware repo is [here](https://github.com/earvingad/sweek-rmk/).

- Clone the repo and run the action. Firmware will be compiled and uploaded when the action finish.
- Flash the central.uf2 to the Left side and the peripheral.uf2 to the right side.
- Edit the keymap as you want with Vial.



Remarks:
- RMK config is only for the matrix version not the diodless.
- Oled screen is not yet supported in RMK.
- RGB is not yes supported in RMK.
- Encoders are buggy.
- Vial works correctly.
